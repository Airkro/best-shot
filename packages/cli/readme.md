## Installation

```bash
npm install @best-shot/cli --save-dev
```

## Usage

```bash
npx -c best-shot <command> [options]
```

## Commands

### watch

Same as `dev` command in watch mode.

### dev

Bundle files in development mode.

### prod

Bundle files in production mode.

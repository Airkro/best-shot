# @best-shot/dev-server

DevServer support of `@best-shot/cli`.

[![npm][npm-badge]][npm-url]
[![license][license-badge]][github-url]
![node][node-badge]

## Installation

```bash
npm install @best-shot/dev-server --save-dev
```

## Usage

```bash
best-shot serve [options]
```
